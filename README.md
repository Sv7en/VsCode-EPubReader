# EPUB Reader

[English](README_EN.md)

一个在 VS Code 中阅读 EPUB 电子书的插件。

github: https://github.com/Sv7en/VsCode-EPubReader.git

## 功能特性

- **侧边栏书架** — 点击左侧活动栏的书本图标即可打开书架，随时浏览和打开书籍
- **6 种主题预设** — 一键切换暗绿、明亮、羊皮纸、护眼绿、夜间、古典黄等阅读主题
- **自定义配色** — 通过颜色选择器自由调整字体颜色、背景颜色和字体大小
- **阅读进度** — 自动保存每本书的阅读位置，下次打开自动恢复
- **目录跳转** — 阅读器顶部下拉菜单可选择章节直接跳转
- **键盘翻页** — 按 `A` 键上一页，`D` 键下一页
- **鼠标滚轮翻页** — 滚轮上下滚动即可翻页
- **自定义书籍目录** — 通过 VS Code 设置指定 EPUB 文件存放路径

## 使用方法

### 打开书架

点击左侧活动栏的书本图标，或按 `Ctrl+3`（Mac 上 `Cmd+3`）聚焦到侧边栏书架。

### 切换主题

在侧边栏书架下方，点击任意主题预设按钮（每个按钮显示颜色预览和名称），已打开的阅读器会立即生效。也可以用预设下方的颜色选择器精细调整字体颜色和背景颜色。

### 自定义书籍目录

默认从插件内置的 `book/` 目录加载书籍。如需使用自己的目录：

1. 打开 VS Code 设置（`Ctrl+,`）
2. 搜索 `EPUB Reader`
3. 将 `reader.bookDirectory` 设置为你的 EPUB 文件夹路径

## 快捷键

| 按键 | 功能 |
|------|------|
| `A` | 上一页 |
| `D` | 下一页 |
| `Ctrl+3` / `Cmd+3` | 打开侧边栏书架 |
| 鼠标滚轮向上 | 上一页 |
| 鼠标滚轮向下 | 下一页 |

## 主题预设

| 预设 | 背景色 | 字体色 |
|------|--------|--------|
| 暗绿 | `#1e1e1e` | `#6a9955` |
| 明亮 | `#ffffff` | `#000000` |
| 羊皮纸 | `#f5eddc` | `#5b4636` |
| 护眼绿 | `#c7edcc` | `#2b4c2b` |
| 夜间 | `#000000` | `#888888` |
| 古典黄 | `#e8d5b7` | `#3e3b36` |

## 配置项

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `reader.fontSize` | 数字 | 14 | 阅读字体大小 (px) |
| `reader.fontColor` | 字符串 | `#6a9955` | 阅读字体颜色 |
| `reader.backgroundColor` | 字符串 | `#1e1e1e` | 阅读背景颜色 |
| `reader.bookDirectory` | 字符串 | `""` | 自定义书籍目录路径（留空则使用内置 `book/` 目录） |
